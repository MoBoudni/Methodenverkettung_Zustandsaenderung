# Methodenverkettung_Zustandsaenderung

Ein Java-Projekt zur Demonstration von Methodenverkettung und Zustandsänderungen mit einer JavaFX-Benutzeroberfläche.

---

## Beschreibung

Dieses Projekt zeigt, wie man Methodenverkettung (Method Chaining) und Zustandsänderungen in Java verwendet. Es simuliert ein Auto, dessen Zustand (z. B. Geschwindigkeit, Motorstatus, Tankstand) durch aufeinanderfolgende Methodenaufrufe geändert werden kann. Die Anwendung nutzt JavaFX für die grafische Benutzeroberfläche.

---

## Features

- **Methodenverkettung**: Demonstriert, wie Methodenaufrufe verkettet werden können.
- **Zustandsänderungen**: Zeigt, wie der Zustand eines Objekts durch Methodenaufrufe geändert wird.
- **JavaFX GUI**: Interaktive Benutzeroberfläche zur Steuerung des Autos.

---

## Voraussetzungen

- Java 17
- Maven
- JavaFX SDK 21.0.9
